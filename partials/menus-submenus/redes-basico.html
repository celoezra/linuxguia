REDES BÁSICAS NO LINUX

Verificando interfaces de rede

O primeiro passo é identificar quais interfaces de rede estão ativas no seu sistema:

ip addr

O comando acima lista interfaces como eth0, wlan0 ou enp3s0.

Configurando IP estático (método clássico)

Edite o arquivo de configuração de interfaces:

sudo nano /etc/network/interfaces

Exemplo de configuração estática para a interface eth0:

iface eth0 inet static
    address 192.168.1.100 #Ip do seu computador-cliete
    netmask 255.255.255.0 #Mascará da internet
    gateway 192.168.1.1 #Gateway da internet
    dns-nameservers 8.8.8.8 1.1.1.1 #Dns para consulta, esse dns é do google e Cloudflare
A linha dns-nameservers é opcional, use servidores confiáveis (Google, Cloudflare, seu provedor).

Aplicando alterações

Reinicie o serviço de rede sem precisar reiniciar o sistema:

sudo systemctl restart networking

Testando conectividade

Para verificar se a rede está funcionando:

ping 8.8.8.8 -c 4
ping google.com -c 4

Se o ping por IP funcionar mas o por nome não, revise suas configurações de DNS.

Gerenciadores mais novos

Em distribuições recentes, pode ser que sua rede seja gerenciada pelo netplan ou pelo NetworkManager:

  • Netplan: configs em /etc/netplan/*.yaml
  • NetworkManager: configs via nmtui (interface de terminal) ou nmcli (linha de comando)

Dica

O nome da interface (eth0, ens33, wlan0) pode mudar dependendo do hardware. Use ip addr para confirmar.
