Como navegar no site

  • Menu superior: os tópicos ficam no menu no topo. Em telas pequenas, use o botão hambúrguer (três barrinhas) para abrir/fechar.
  • Submenus: alguns itens (como REDES) têm setinha e abrem um submenu com mais páginas (Básico, Comandos úteis, Troubleshooting).
  • Âncoras: cada página possui um id (por exemplo, #comandos-top). Os links do menu levam direto para a seção correta.
  • Tema claro/escuro: clique em “Alternar Tema” no topo para trocar entre Dark e Light.
  • Fechar o menu: toque fora do menu (no fundo escurecido) ou pressione Esc para fechar rapidamente no mobile.
  • Rolar para o conteúdo: ao clicar num item, a página rola suave para o início da seção; o conteúdo é focado para melhorar a acessibilidade.
